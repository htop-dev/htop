# Security Policy

## Reporting a Vulnerability

To report a security vulnerability in htop, send an email to htop-maintainers@groups.io.

Alternatively, you may submit vulnerability reports through [GitHub Security Advisories](https://github.com/htop-dev/htop/security/advisories/new).

Please include affected versions, impact, and reproduction steps when possible. If you want a private reply, provide a contact method.

We will acknowledge reports as soon as practical.

## Bug bounty

There is no bug bounty program.
